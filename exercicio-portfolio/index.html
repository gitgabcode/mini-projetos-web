Meus projetos

Cyber Virus History

Cyber Virus History é um site educacional sobre a evolução dos vírus de computador, desde o Creeper até ameaças como LoveLetter e Heartbleed. Desenvolvido por Gabriel Marcelino, o projeto utiliza Flexbox para o menu de navegação, gradientes lineares no fundo e links internos para facilitar a navegação. Também inclui um botão fixo que aparece ao passar o mouse, levando o usuário de volta ao topo da página. O site foi construído com HTML semântico e CSS para uma estrutura acessível e fácil de manter.

Cordel Nordeste Encantado

Neste projeto, Cordel Nordeste Encantado, meu objetivo foi treinar o uso das propriedades relacionadas ao background em CSS, como background-image, background-position, background-size e background-repeat, aplicando-as para criar um design visualmente atraente. Também foquei em treinar o uso de position para posicionar elementos de forma precisa e dinâmica. O design do projeto foi inspirado pela cultura nordestina, utilizando imagens de paisagens locais e cores que remetem a folhas antigas e xilogravuras, criando uma experiência imersiva e poética.
